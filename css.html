Mise en Forme XML avec CSS

Entrez un contenu XML et un style CSS dans les champs ci-dessous, puis cliquez sur "Mise en forme" pour afficher le résultat stylisé dans l'iframe. Exemples :

Exemple XML :

Exemple CSS :

    margin-bottom: 30pt;
}
montant {

}
